================================================================================

- Create templates for each day of the week (Monday-Sunday)
- Set different quests for different days
- Manage rewards (XP and coins) per quest
- See live preview while editing
- Easily modify existing templates

================================================================================
HOW TO USE
================================================================================

FOR ADMINS:
-----------
1. Go to Admin Dashboard → Gamification → Quest Templates
2. See 7 cards (one for each day of the week)
3. Click Edit on any day to open the form
4. Fill in quest details:
   - Title (quest name)
   - Type (select from dropdown)
   - Description (instructions)
   - Target (number to achieve)
   - Reward XP (experience points)
   - Reward Coins (optional currency)
5. Check the live preview
6. Click Save Template
7. Done! Template is now active

FOR DEVELOPERS:
---------------
Get Monday's template:
  $quest = DailyQuest::getByDayOfWeek(0);

Get all templates:
  $templates = DailyQuest::getWeeklyTemplates();

Auto-generate daily quest:
  $template = DailyQuest::getByDayOfWeek(now()->dayOfWeek);
  if ($template) {
    DailyQuest::create([...]);
  }
